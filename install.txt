Please follow the steps below to install the module:
1. Install Microsof Sql Server http://www.microsoft.com/en-us/server-cloud/products/sql-server/
2. Install Php Sqlsrv Driver (http://php.net/manual/en/sqlsrv.installation.php)
3. Copy all files/folders under magento folder to Magento Root to install module
Example: Our Magento Root /var/www/magento
The module folder /home/ubuntu/magento-sqlsrv-adapter
We did the following command to copy and replace:
cp -rf /home/ubuntu/magento-sqlsrv-adapter/magento/* /var/www/magento/
4. Create a database in Microsoft Sql Server
5. Import the sql file script.sql into the database created in the step 4.
6. Open app/code/community/Salore/Sqlsrv/etc/config.xml and change connection credential
Example:
				<connection>
					<host><![CDATA[192.168.1.99]]></host>
					<dbname><![CDATA[salore]]></dbname>
					<username><![CDATA[sa]]></username>
					<password><![CDATA[123456]]></password>
					<type><![CDATA[sbsqlsrv]]></type>
					<active>1</active>
				</connection>
7. Clear all magento caches and test new order
